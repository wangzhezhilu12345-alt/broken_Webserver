V3 测试页已经就位

这个首页主要给你快速验证当前 HTTP 服务能不能正常返回静态内容，以及 404、504 这类错误页是否已经切到自定义页面。

刷新首页 触发真实 404 查看 504 页面
404 Not Found 访问一个不存在的地址，验证服务是否返回自定义 404 页面和正确状态码。 /missing-page 404 Static Preview 直接进入 404 页面模板，用来检查样式、文案和布局细节。 /404.html 500 Internal Error 预留服务内部错误页，方便后续统一扩展异常处理体验。 /500.html 504 Gateway Timeout 网关超时页，可用来测试代理层超时后的前端展示效果。 /504.html

建议测试方式

  • 先访问首页，确认静态资源和布局正常。
  • 再点 /missing-page，确认浏览器显示的是 HTML 错误页，不是纯文本。
  • 直接打开 /504.html，确认状态码和页面内容都符合预期。
